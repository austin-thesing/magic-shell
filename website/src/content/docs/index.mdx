## Quick Start

```bash
# Install Magic Shell
curl -fsSL https://raw.githubusercontent.com/austin-thesing/magic-shell/main/install.sh | bash

# Configure your API key
msh --setup

# Try your first command
msh "find all files larger than 100MB"
```
